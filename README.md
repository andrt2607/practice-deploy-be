## Run Locally

Clone the project

```bash
  git clone https://github.com/andrt2607/practice-deploy-be
```

Go to the project directory

```bash
  cd practice-deploy-be
```

Install dependencies

```bash
  npm install
```
or
```bash
  yarn install
```

Start the server

```bash
  npm run dev:start
```
or
```bash
  yarn run dev:start
```
